Amazon Clone

This project is a static clone of the popular e-commerce website, Amazon. Built using only HTML and CSS, this clone replicates the core layout and design elements of the original site.

Key Features:

  • Responsive Design: The layout adapts to various screen sizes, ensuring a seamless shopping experience on both desktop and mobile devices.
  • Product Showcase: A visually appealing grid of product cards, each featuring an image, title, price, and a button for more details.
  • Navigation Bar: An easy-to-use navigation menu that allows users to browse through different categories and access the homepage.
  • Search Bar: A functional search bar at the top of the page, designed to mimic the original Amazon interface.
  • Footer: Contains links to customer service, privacy policy, and social media icons, enhancing user navigation. Technologies Used:

HTML for structure and content organization.
CSS for styling and layout, ensuring an aesthetically pleasing design.
